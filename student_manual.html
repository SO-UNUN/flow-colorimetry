ส่วนที่ 2หลักการทำงาน

Digital Image Colorimetry

กล้องมือถือเป็นเซนเซอร์รับแสง 3 ช่อง (Red, Green, Blue) ในแต่ละพิกเซล เมื่อสารละลายมีสี → แสงผ่านเปลี่ยนไป → ค่า RGB เปลี่ยน → เราใช้การเปลี่ยนแปลงนี้บอกความเข้มข้นได้

RGB intensity → Absorbance → Concentration R, G, B = 0–255 ต่อ channel
A = -log₁₀(I / I₀)
C = (A - intercept) / slope

RGB vs HSV — ต่างกันอย่างไร?

ระบบสีหมายถึงเหมาะใช้เมื่อ
R (Red)ความเข้มแสงสีแดงสารละลายดูดกลืนสีแดง (สารสีฟ้า/เขียว)
G (Green)ความเข้มแสงสีเขียวสารละลายดูดกลืนสีเขียว (สารสีม่วง/แดง)
B (Blue)ความเข้มแสงสีน้ำเงินสารละลายดูดกลืนสีน้ำเงิน (สารสีเหลือง/ส้ม)
H (Hue)เฉดสี (0-360°)บอก "สีอะไร" ของสารละลาย
S (Saturation)ความอิ่มตัว (0-100%)ความเข้มข้นของสีนั้น
V (Value)ความสว่าง (0-100%)ความสว่างโดยรวม
💡
เคล็ดลับ: สำหรับสารละลายสีม่วง (เช่น KMnO₄) → channel G มักเปลี่ยนชัดที่สุด เพราะสารละลายดูดกลืนแสงเขียวได้ดี (complementary color)

ส่วนที่ 3การเตรียมการถ่ายวิดีโอ

คุณภาพของผลการวิเคราะห์ขึ้นกับคุณภาพของวิดีโอที่ถ่ายมาก — ทำตามคำแนะนำเหล่านี้

📱 ตั้งค่ากล้องมือถือ

1️⃣ ใช้แอปกล้อง manual mode เช่น ProCam (iOS), Open Camera (Android), หรือ Camera FV-5

2️⃣ ล็อค exposure (ปิด auto exposure) — ถ่ายเฟรมแรกที่สว่างพอแล้วล็อคไว้

3️⃣ ล็อค white balance (ปิด auto WB) — สำคัญมาก! ถ้าไม่ล็อค กล้องจะ "ปรับ" สีให้ดูปกติ ทำให้ค่าจริงผิดเพี้ยน

4️⃣ ล็อค focus ที่ระยะ flow cell

5️⃣ ความละเอียด 720p หรือ 1080p เพียงพอ (ไม่ต้อง 4K ไฟล์ใหญ่ใช้เวลานาน)

6️⃣ Frame rate 30 fps เพียงพอ (ไม่ต้อง 60 fps)

💡 การจัดแสง

1️⃣ ใช้ light box หรือกล่องปิดมิดที่มีแสง LED คงที่

2️⃣ แสงต้องผ่าน flow cell โดยตรง (transmission mode)

3️⃣ หลีกเลี่ยงแสงแดด และแสงแวดล้อมที่เปลี่ยน (ตู้ปฏิบัติการ ห้องเรียน)

4️⃣ พื้นหลังควรเป็นสีขาวหรือดำที่สม่ำเสมอ

📐 การวาง flow cell + กล้อง

1️⃣ ใช้ ขาตั้งกล้อง หรือฐานยึดมือถือ — ห้ามถือมือ!

2️⃣ ระยะกล้องถึง flow cell คงที่ ตลอดการทดลอง

3️⃣ ตำแหน่งของ flow cell ในเฟรมไม่เปลี่ยน (ไม่ขยับแม้แต่นิด)

4️⃣ ระยะ macro ใกล้พอที่จะเห็น flow cell ชัดเจน

เลือกโหมดที่เหมาะกับงานของคุณใน sidebar ด้านซ้าย

Core
RGB/HSV ตามเวลา — ใช้งานทั่วไป รวดเร็ว เหมาะกับการดูภาพรวม
Core + Plus
Core + smoothing + ตรวจหา plateau อัตโนมัติ — เหมาะกับ flow ที่มีหลายความเข้มข้น
Advanced
Plus + Beer-Lambert + Calibration curve — เหมาะกับงานวิจัย วัด unknown สาร
📊 Core (โหมดพื้นฐาน)

เหมาะสำหรับ: นักศึกษาที่อยากดูว่าค่า RGB เปลี่ยนตามเวลาอย่างไร, สำรวจข้อมูลเบื้องต้น

สิ่งที่ได้: กราฟ R, G, B vs time + HSV (ถ้าเปิด)

📈 Core + Plus (เพิ่ม smoothing + plateau)

เหมาะสำหรับ: flow ที่มีหลายความเข้มข้นต่อเนื่อง — ต้องการแยกแต่ละความเข้มข้นออกมา

สิ่งที่ได้เพิ่ม: เส้นกราฟที่ smooth + ตรวจหา plateau (ช่วงค่าคงที่) + ตารางค่าเฉลี่ยแต่ละ plateau

🔬 Advanced (วิจัย + calibration)

เหมาะสำหรับ: งานวิจัยที่ต้องสร้าง calibration curve และคำนวณ unknown concentration

สิ่งที่ได้เพิ่ม: Absorbance plot + ป้อน known concentrations → Linear regression → predict unknown

ส่วนที่ 6การอ่านผลและการวิเคราะห์

กราฟ RGB — แต่ละสีหมายความว่าอะไร?

🔴 เส้นสีแดง (R) ลดลง = สารละลายดูดกลืนแสงสีแดงมากขึ้น

🟢 เส้นสีเขียว (G) ลดลง = สารละลายดูดกลืนแสงสีเขียวมากขึ้น

🔵 เส้นสีน้ำเงิน (B) ลดลง = สารละลายดูดกลืนแสงสีน้ำเงินมากขึ้น

เลือก channel ที่เปลี่ยนแปลงมากที่สุดเป็นตัวแทนการวิเคราะห์

Plateau — ทำไมต้องตรวจหา?

ถ้าทดลองมีหลายความเข้มข้นต่อเนื่อง (เช่น 0.1 M → 0.2 M → 0.3 M ตามลำดับ)

ค่าจะคงที่ชั่วครู่เมื่อสารละลายแต่ละความเข้มข้นเข้ามาเต็มใน flow cell

โปรแกรมจะตรวจหาช่วงที่ค่าคงที่นี้ → คำนวณค่าเฉลี่ย → ใช้เป็นจุดข้อมูล calibration

Calibration Curve — Advanced mode

A = slope × C + intercept เมื่อ slope สูง = sensitivity สูง
R² ใกล้ 1 = linear relationship ดี
📊 ตีความค่า R²

🟢 R² ≥ 0.99 — Excellent (ใช้ได้เลย)

🔵 R² ≥ 0.95 — Good (ใช้ได้ในงานทั่วไป)

🟡 R² ≥ 0.90 — Fair (ตรวจสอบความผิดพลาด)

🔴 R² < 0.90 — Poor (ทำซ้ำ)
